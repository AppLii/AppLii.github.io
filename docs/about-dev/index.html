主な使用言語・開発環境など

私たちのチームは、最新の技術と効率的なツールを駆使して、高品質なソフトウェアの開発を目指しています。以下に、私たちの主な技術スタックを紹介します。

プログラミング言語

Python：データ分析、機械学習、バックエンド開発に広く使用されています。

JavaScript：フロントエンド開発でのインタラクティブな要素の作成に不可欠です。

フロントエンド開発

Vue.js：ユーザーにやさしいインタラクティブなウェブインターフェースの構築に使用。

HTML, CSS, SCSS：ウェブページの構造とスタイルを定義します。

バックエンド開発

Flask, FastAPI：軽量でパワフルなAPIの開発に適しています。

Node.js：非同期イベント駆動型のアプリケーション開発に使用されます。

クラウド・サービス

Microsoft Azure：高可用性のアプリケーションデプロイとサービスの実行に利用。

コンテナ技術

Docker：アプリケーションのデプロイメントとスケーリングを容易にします。

バージョン管理システム

GitHub, GitHub Actions：コードの管理と自動化されたワークフローに使用。

オペレーティングシステム

Linux：サーバーと開発環境の安定性に優れています。

ハードウェア

Raspberry Pi：小規模なプロジェクトや教育用途に適しています。

インタラクティブ開発環境

Jupyter Notebook：データサイエンスと機械学習プロジェクトでのコードの実行と文書化に最適。
